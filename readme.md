> Chesterish Theme for [Hyper](https://hyper.is) based on [Jupyter](https://github.com/dunovank/jupyter-themes) schemes.

![](screen.png)
![](screen_tabs.png)


## Install

Add `hyper-chesterish` to the plugins array in your `~/.hyper.js` config.
